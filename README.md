# ⚽ FIFA World Cup Analytics Dashboard

An interactive football analytics and match prediction system built with Streamlit, featuring comprehensive World Cup statistics, team performance analysis, and machine learning-based match outcome predictions.

## 🚀 Features

### 📊 Tournament Overview
- Total matches, teams, and average goals statistics
- Interactive timeline showing matches per year
- Quick overview of tournament history

### 🌍 Team Analytics
- Detailed statistics for any World Cup team
- Head-to-head team comparison
- Match count, goals scored, and win rate metrics

### 📈 Goal Trends
- Visualization of average goals per World Cup
- Identification of highest-scoring tournaments
- Trend analysis across tournament history

### 🥊 Head-to-Head
- Compare any two teams' historical matchups
- Complete match history between selected teams
- Interactive data display

### 🤖 Model Performance
- Machine learning model accuracy metrics
- Feature importance visualization
- Confusion matrix for prediction analysis
- Random Forest classifier implementation

### 🔮 Match Predictor
- Predict match outcomes between any two teams
- Probability distribution for home win, draw, and away win
- Detailed explanation of prediction factors
- Real-time prediction based on historical performance

### 🏅 Team Rankings
- Comprehensive team rankings
- Win rate, matches played, and goals scored
- Top 10 teams visualization

### 📰 Recent Matches
- Latest FIFA World Cup matches
- Quick access to recent results

## 🛠️ Technologies Used

- **Python 3.8+**
- **Streamlit** - Web application framework
- **Pandas** - Data manipulation and analysis
- **NumPy** - Numerical computations
- **Plotly** - Interactive visualizations
- **Scikit-learn** - Machine learning models
  - Random Forest Classifier
  - Train-test split
  - Accuracy metrics
  - Confusion matrix
